# Sales Dashboard Project

## Project Overview
This project is a Sales Dashboard that helps analyze sales performance,
profit, and customer trends using data visualization.

## Tools Used
- Power BI / Python
- Excel
## Features
- Total Sales Analysis
- Profit Analysis
- Region wise Sales
- Monthly Sales Trend
- Customer Segment Analysis

## Dataset
The dataset contains information about:
- Order ID
- Product
- Sales
- Profit
- Region
- Customer Segment

## Dashboard Insights
- Identify top performing regions
- Find high profit products
- Analyze monthly sales trends

## How to Use
1. Download the dataset
2. Open Power BI file / Run Python script
3. Explore the dashboard
